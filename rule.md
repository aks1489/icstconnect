# Agent Rules

## Documentation Synchronization
1. **Always Update Markdown Files:** Whenever the AI agent modifies, adds, or structural changes the codebase, it MUST proactively update any related markdown documentation files (such as those in the `/docs` folder, `README.md`, or component-specific `.md` files). 
2. The documentation must stay strictly in sync with the current state of the codebase at all times.
3. Before finishing a task, the agent must verify that all newly implemented features, changed APIs, or modified user interfaces are fully documented in the respective `.md` files.
