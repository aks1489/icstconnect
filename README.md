## Project Context: ICST Connect

This serves as the central management system for our institution. 

### Key Features (Planned/In-Progress)
- **Financial System**: comprehensive balance sheet transaction tracking (Assets, Liabilities, Income, Expenses).
- **Student Fees Management**: 
    - Individual fee structures per student.
    - Support for discounts (Admission & Total).
    - Flexible payment types (One-time vs Monthly).
    - Student-friendly "gentle" payment reminders.
    - Admin dashboard for fee oversight.
